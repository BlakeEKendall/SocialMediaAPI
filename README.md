# SocialMediaAPI
24-hour project- social media app

Built by: Team Blue (Blake Kendall, Michael Milroy, and Karras Pierce)
For: Eleven Fifty Academy Blue Badge 24-hr Project
Goal: Create a WebAPI that mimics Social Media functionality with Posts, Comments, Replies, and Likes
Postman utlized to create/authorize users and get authorization tokens
Swagger used to access CRUD methods for Posts, Comments, Replies, and Likes
